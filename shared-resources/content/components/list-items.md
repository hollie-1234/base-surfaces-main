# List Item Component — Writing Guidelines

## Title Guidelines

### Rules
- **Length:** Just a few words
- **Structure:** Put the most important words first
- **Purpose:** Summarize or introduce the content
- **Tone:** Statement rather than question (unless asking the user something)
- **Formatting:** Sentence case (only capitalize the first letter of the first word)
- **Punctuation:** No full stop

### Examples

#### ✓ Do
- "Payment methods"
- "Security and privacy"
- "Add a currency"
- "Direct Debits"

#### ✗ Don't
- "How would you like to pay?" — Question format (unless contextually appropriate)
- "Payment Methods" — Title case
- "Add a currency." — Has full stop

## Subtitle/Description Guidelines

### Rules
- **Length:** Maximum two sentences; account for translation expansion
- **Format:** Plain text — no bold text, links, or fancy formatting
- **Punctuation:** Include a full stop
- **Consistency:** If one item in a set has a subtitle, include subtitles for all items

### Examples

#### ✓ Do
- "Change your security and privacy settings."
- "Add money to your account with a bank transfer."

#### ✗ Don't
- Mix items with and without subtitles in the same list
- Use formatting or inline links

## Additional Information Guidelines

### When to Use
- Apply only after incorporating a description
- Use sparingly and strategically

### Rules
- **Length:** Maximum two lines; consider translation impact
- **Links:** Due to accessibility constraints, only 1 link allowed
- **Link placement:** Must be appended to content

## Values Guidance

When displaying numerical amounts or values:

### Rules
- Use for supporting numerical data
- Maintain single-line display to prevent wrapping
- Apply sentiment colors (positive/negative indicators) alongside text
- Never rely solely on color for accessibility

### Examples
- Transaction amounts
- Account balances
- Status indicators with values

## Consistency Rule

Maintain visual hierarchy consistency across list items. If one item has certain elements (subtitle, additional info, value), consider whether others in the set should match to avoid appearing incomplete.
